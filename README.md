# MashaAllah Enterprise

# Description

MashaAllah Enterprise is a modern and high-performance E-Commerce web application built with Django. The platform allows users to browse products, add items to the cart, and place orders **without creating an account**, ensuring a fast and user-friendly shopping experience.

# Features

### Shopping Features
- Browse products easily
- Product detail view
- Add to cart without login
- Guest checkout system
- Session based cart storage
- Quick order placement

### UI & Performance
- Clean and modern design
- Fully responsive layout
- GSAP powered animations
- Fast loading performance
- Mobile friendly interface

### Admin Features
- Product management
- Order management
- Customer order tracking
- Image upload support
- Secure admin dashboard

## Core Functional Highlights

- Guest User Order System  
- Session-Based Cart Management  
- Dynamic Product Display  
- Optimized Database Structure  
- Scalable Django Architecture  
- Animation Integrated Frontend

# Installation

- git clone https://github.com/programmerrm/mashaallahenterprise.git

### Windows
    - python -m venv env
    - env\Scripts\activate
    
### Linux / Mac
    - python3 -m venv env
    - source env/bin/activate

- pip install -r requirements.txt
- python manage.py makemigrations
- python manage.py migrate
- python manage.py createsuperuser
- python manage.py runserver
- http://127.0.0.1:8000/
- http://127.0.0.1:8000/admin
